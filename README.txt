# Multiple Year Stock Data

Multiple years tock data is a file containing stock data. The subroutine stockloop loops throught data and finds totals of values, percent increases, and maxima/minima.

## Installation

Copy the code into your desired spreadsheet. 

## Usage

The spreadsheet should have these columns: ticker, opening, closing, stock volume.Each year's data should be in a separate tab. In module 2, subloop worksheet replace 2018,2019, and 2020 with your tab names.
